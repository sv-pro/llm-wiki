# Learning Objectives

- Understand how scope creep and privilege growth emerge in agentic systems that combine autonomy, tool access, and repeated approvals.
- Learn how observed execution traces can be turned into bounded capability profiles and manifest drafts for least-privilege enforcement.
- See how declarative manifests can encode action boundaries, trust levels, taint rules, provenance constraints, and approval gates as deterministic runtime policy.
- Leave with a practical pattern for securing tool-using agents through policy-driven boundaries rather than reactive filtering alone.
