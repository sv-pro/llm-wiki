# Agent Hypervisor — Documentation

**For the complete documentation map, see [INDEX.md](INDEX.md).**

This directory contains the research, architectural, and reference documentation for the Agent Hypervisor architecture. It is organized around five categories:

- **Canonical concepts** — the core thesis, whitepaper, and architectural standards
- **Threat model & security analysis** — formal threat boundaries and comparisons
- **Standards & manifest ideas** — 12-Factor Agent, ADRs, glossary
- **Article drafts** — *The Missing Layer* series and planned *Taint* series (in [pub/](pub/))
- **Summit & talks** — presentation decks and interactive demo (in [../presentation-core/](../presentation-core/), etc.)
- **Implementation reference** — PoC codebase documentation
